## Dataset Access
The dataset used in this project (120 years of Olympic athlete results) is publicly available and can be downloaded from the following link:

https://www.kaggle.com/datasets/heesoo37/120-years-of-olympic-history-athletes-and-results

Due to GitHub file size limitations, the full CSV file is not stored in this repository. 
To run the notebook, download the dataset from the link above and place `athlete_events.csv` in the same directory as the notebook.
